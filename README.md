# Sistema Web para Pedidos de Comida Local - Cusco

## Descripción del Proyecto
[cite_start]Este repositorio contiene la estructura inicial para el desarrollo de un sistema web destinado a gestionar pedidos de comida local para una startup en la ciudad de Cusco. [cite_start]Este proyecto se desarrolla como parte del **Laboratorio 1: Introducción al entorno de desarrollo y Git**[cite: 1].

## Objetivo
[cite_start]El propósito de este repositorio es establecer la configuración de un entorno de desarrollo moderno y aplicar buenas prácticas de control de versiones utilizando Git en un proyecto básico[cite: 3, 6].

## Herramientas Utilizadas
Para la configuración inicial y gestión del código se han empleado las siguientes herramientas:
* [cite_start]**Control de Versiones:** Git [cite: 13]
* [cite_start]**Plataforma de Alojamiento:** GitHub [cite: 14]
* [cite_start]**Entorno de Desarrollo Integrado (IDE):** Visual Studio Code / IntelliJ [cite: 12]
